# Q&A Support Bot (RAG, Embeddings, Vector DB)

## Overview
This project builds a Q&A bot that answers questions using Retrieval Augmented Generation (RAG). It crawls a website, chunks text, generates embeddings, stores them in a vector database, and serves answers via an API.

## Tech Stack
- Python
- FastAPI
- OpenAI API (embeddings + GPT)
- FAISS (vector database)
- BeautifulSoup (web crawling)
